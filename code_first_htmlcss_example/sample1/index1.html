Visual Studio Code는 https://code.visualstudio.com/ 에서 다운로드가 가능합니다.

Visual Studio Code를 설치하시면, 이제 몇가지 확장 프로그램을 더 설치하시면 웹 개발을 빠르게 할 수 있습니다.

프리프로세서(Preprocessor)란

프리프로세서는 번역하자면 “선행처리기” 라고 하는데, 주로 프로그래밍 언어에서 사용됩니다.
원래 CSS나 HTML 파일에는 변수 또는 함수 같은 코드가 없어서, 사이트의 규모가 커지고, 복잡도가 증가하게 되면, 일일이 모든 코드를 입력하는 것이 힘들 수 있습니다.
이때 SASS와 같은 CSS 프리프로세서를 사용하면, 변수, 함수 및 값의 연산 등을 사용할 수 있게 되어, CSS의 개발이 보다 체계화되고, 빠르게 개발을 할 수 있습니다. 다만 원시 코드를 그대로 브라우저에서 사용할 순 없고, 컴파일 과정을 통해 CSS로 변환해야만 하기 때문에, 번거로울 수 있지만, 이 번거로움을 상쇄하는 강력한 기능이 있기 때문에, 최근 많은 개발자들이 사용하고 있습니다. 필자 또한 웹 개발용으로 SASS를 강력 추천합니다.
