고정형 line-height

텍스트 속성 중 line-height는 문단과 문단 사이의 간격을 조절하는데 사용되는 속성입니다.
여기서 line-height를 따로 다루는 이유는 font-size와도 관계가 있습니다. font-size 인 경우 고정형과 가변형이 있다고 학습하였습니다.
line-height 인 경우에도 고정형과 가변형 속성 둘 다 적용가능하지만, 필자는 되도록 line-height 인 경우 가변형을 적용하라고 권하겠습니다.
